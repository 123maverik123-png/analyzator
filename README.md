# 📞 Анализатор разговоров

**Анализатор разговоров** – это десктопное приложение для автоматической расшифровки и анализа аудиозаписей звонков технической поддержки с использованием **Whisper** (распознавание речи) и **YandexGPT** (смысловой анализ). Программа помогает операторам быстро оценивать содержание разговоров, фиксировать проблемы и получать статистику.

---

## 🚀 Возможности

- **Автоматическое распознавание речи** (Whisper: tiny, base, small, medium, large) с выбором модели и устройства (CPU/GPU).
- **Смысловой анализ** через YandexGPT: определение проблемы, категории, резюме, факта решения.
- **Диаризация** (разделение голосов оператора и абонента) – опционально.
- **Тональность** разговора (негатив/нейтраль/позитив) – локально, через ruBERT.
- **Геймификация**: достижения, уровни, очки опыта (XP) за обработанные и решённые звонки.
- **Лицензирование** с демо-периодом (14 дней) и активацией по ключу.
- **Настройки приложения**:
  - выбор модели Whisper (tiny, base, small, medium, large);
  - выбор устройства (CPU / GPU NVIDIA 30XX+);
  - смена папки для записей;
  - светлая/тёмная тема.
- **Встроенный аудиоплеер** с управлением (воспроизведение, перемотка, положение).
- **Автообновление** через GitHub (проверка при запуске).
- **Статистика и дашборд** на центральном сервере (для руководителей).
- **Экспорт данных** в CSV.

## 📥 Установка

### Для конечного пользователя
1. Скачайте установщик `MyCallAnalyzerSetup.exe`.
2. Запустите его (требуется интернет для скачивания Python, VLC и VC++ Redistributable).
3. Следуйте инструкциям мастера установки.
4. После установки запустите приложение через ярлык на рабочем столе или в меню «Пуск».

> **Примечание:** установщик скачивает необходимые компоненты автоматически, поэтому размер дистрибутива минимален.

---

## ⚙️ Первый запуск

1. При первом запуске введите свои фамилию и имя (это будет вашим идентификатором оператора).
2. Укажите папку, в которую сохраняются аудиозаписи звонков (кнопка **«⚙ Папка записей»**).
3. Бот запустится автоматически и начнёт отслеживать новые файлы в выбранной папке.

---

## 🖥️ Интерфейс

### Главное окно
- **Вкладка «Мои разговоры»** – список всех обработанных звонков. Новые выделены жирным шрифтом. При выборе звонка справа отображаются:
  - резюме (кнопка копирования);
  - проблема (кнопка копирования);
  - расшифровка с метками ролей (если включена диаризация);
  - аудиоплеер для прослушивания записи.
- **Вкладка «Достижения»** – список бейджей и прогресс по ним, а также уровень и XP.

### Верхняя панель
- **«📋 Логи»** – открывает окно с подробными сообщениями от бота.
- **«⚙ Папка записей»** – выбор папки для отслеживания.
- **«⚙ Настройки»** – выбор модели Whisper и устройства (CPU/GPU).
- **«🔄 Проверить обновления»** – ручная проверка новой версии.
- **«🔄 Перезапустить бота»** – перезапуск фонового процесса распознавания.
- Переключатель темы (светлая/тёмная).

### Нижняя панель
- Статус лицензии (демо-период, подписка, истекла).
- Ссылка «Продлить подписку» для ввода лицензионного ключа.
- Версия программы и контактный email.

---

## 🔐 Лицензирование

- При первом запуске автоматически активируется **демо-период** (14 дней).
- По окончании демо-периода требуется ввод лицензионного ключа.
- Ключ можно получить у разработчика или через административный интерфейс сервера.
- Для активации нажмите **«Продлить подписку»** в левом нижнем углу и введите ключ.
- В случае отсутствия интернета программа работает в **grace-периоде** (3 дня) на основе кэшированного статуса.

---

## 🧠 Настройка модели Whisper

Через кнопку **«⚙ Настройки»** вы можете выбрать:
- **Модель**: `tiny`, `base`, `small`, `medium`, `large`. Чем больше модель, тем выше точность, но медленнее работа и больше требований к памяти.
- **Устройство**: `CPU` или `GPU (NVIDIA)`. Для GPU необходима видеокарта уровня NVIDIA серии 30XX или новее, установленный CUDA Toolkit и драйверы.

После изменения настроек перезапустите бота кнопкой **«🔄 Перезапустить бота»**.

    tkinter / sv-ttk – графический интерфейс

    cryptography – шифрование секретов

    PyInstaller – сборка в EXE

    Inno Setup – создание установщика
